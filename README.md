# Portfolio

Portfolio personnel construit avec Next.js 15 et Tailwind CSS.

## Stack Technique

- Next.js 15 (App Router, Turbopack)
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui
- next-themes

## Développement

```bash
pnpm install
pnpm dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Build

```bash
pnpm build
pnpm start
```

## Qualité du Code

```bash
pnpm lint
pnpm type:check
```
